# CS 201 Final Project: Book Application

This is my final project for CS 201. The project is a Book Application that keeps track of books a user has read. The program runs in the command line and gives the user a menu of options.

The application stores two different types of books:

Printed books  
Audiobooks

For every book, the program stores the title, author, genre, and book type. Printed books also store the number of pages. Audiobooks store the duration in minutes.

The program also calculates the cost of each book. Printed books cost $0.50 per page, and audiobooks cost $0.25 per minute.

## How to Run the Program

All of the Java files should be in the same folder.

To compile the program, use:

```bash
javac *.java
